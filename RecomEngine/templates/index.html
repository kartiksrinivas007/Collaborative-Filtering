Movie Prediction

Welcome to the My movie recommendation system, I have built a Collaborative filtering Engine. What this means is that a single rating from a particular user affects the ratings of all the other users who have rated the same movies. There are two ways to implement this, In this system I have used the KNN Algorithm. The second algorithm uses a matrix factorization algorithm (using ML and pytorch) and can be found on the Github repo for this project.

The Dataset

This Model uses the publicly available MovieLens dataset, it has been trained using approximately 10,000 movies. The dataset is divided into three parts, the first part contains information about the movies, the second part contains information about the users and the third part contains the ratings of the users.

Plotting the Dataset

For imlpementing a KNN - algorithm , I need to know the distance between movies.. How do I do that?. The axees are the User ratings! Basically if Ram rated Movie A as 2.5 stars and Shyam rated Movie A as 5 stars, then Movie A would have co-ordinates (2.5,5)! But this dataset has nearly 610 users !
